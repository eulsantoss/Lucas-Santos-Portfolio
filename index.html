  PORTFOLIO  

Seja bem-vindo(a) a área do catálogo de criações! Abaixo você encontra todos os meus projetos já feitos, tanto pessoais quanto profissionais. TODOS os sites abaixo são unicamente de minha autoria no quesito Desenvolvimento, dentre projetos criados do zero (LuMake, StormNews, Biografia) e projetos de Redesign e Reestruturação (Banco do Brasil). Deixo claro que os sites de Redesign e Reestruturação NÃO serão destinados à uso comercial sem autorização dos detentores dos direitos da empresa/marca/instituição.

LuMake

O Projeto LuMake foi destinado à divulgação de vendas de produtos de beleza como um projeto profissional não remunerado no intuito de agregar experiência à mim, de forma que o/a cliente possa desfrutar do conteúdo como uma Landing Page para atrair Leads ao contato direto para realizar suas vendas.

StormNews

O Projeto StormNews foi destinado à uma experiência pessoal para sites voltados para notícias, blogs e sites com acesso à informações do dia-a-dia. Não foi feito para uso comercial, apenas como estudo de modelo de site de notícias.

Banco do Brasil

O Projeto Banco do Brasil foi destinado à experiência pessoal de estudo, sem uso comercial, onde eu aplico estudo de observação, redesign e reestruturação para um modelo mais prático e moderno. O projeto NÃO possui fim de lucratividade ou fim comercial, é apenas como projeto de estudo.

Biografia

O Projeto Biografia foi destinado à experiência pessoal de estudo com sites de biografia e história, adotado o modelo Landing Page com menu de índice para o leitor, onde eu exalto a vida do jogador de futebol profissional Giorgian de Arrascaeta, do time do Flamengo. Não possui fins comerciais, apenas projeto para estudo.
