CONHECIMENTOS

HTML5 é uma linguagem de marcação da qual é responsável por toda estruturação do site, informações e elementos da página. Este elemento é a partida inicial da criação de uma página web.

CSS3 é uma linguagem de design gráfico escrita dentro do código HTML de um site, onde determina estilos, posicionamentos, aparência e design geral da página. Assim como o HTML, o CSS não é uma linguagem de programação.

JavaScript é uma linguagem de programação que pode ser usada para diversas coisas, como criação de app’s, área interna do website (back-end) e diversas outras finalidades. Em conjunto com HTML e CSS, o JavaScript tem finalidade de trazer dinâmica, movimento e inteligência à página
